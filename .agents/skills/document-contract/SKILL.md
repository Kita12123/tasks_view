# When to use
- To think about how to work with AI agents.
- To add or update documents.

# Workflow: Adding a Document
1. **Add frontmatter** with properties (name, description, tags, cover) at the top of the file. See [FRONTMATTER.md](references/FRONTMATTER.md) for all property types.
2. **Write content** using standard Markdown for structure, plus Obsidian-specific syntax below.
3. **Link related documents** using standard Markdown links for external URLs.
4. **Embed content** using standard Markdown syntax for images and PDFs.
5. **Add callouts** for highlighted information using standard Markdown blockquotes.
6. **Verify** the note renders correctly in Obsidian's reading view.
