# When to use
- To think about how to work with AI agents.
- To add or update documents.
- To add or update glossary.

# Workflow: Adding or Modifying Glossary
1. Copy the `GLOSSARY.md` file from the `references` directory to the project root directory. *Only if the `GLOSSARY.md` file does not exist in the project root directory.*
2. Update the `GLOSSARY.md` file in the project root directory to add or update terms and definitions as needed.
3. Refer to the `GLOSSARY.md` file in the project root directory when working with AI agents or when adding or updating documents and coding.
