# My OpenAPI Rule

## Contracts
- Require description for all API endpoints, parameters, schemas, schema properties, request bodies, and responses.
- The description for all API endpoints should include a bulleted list of detailed information about what is being done.
- Require operationId to be in camelCase and unique across the specification.
