# OpenAPI規約

## 契約
- 全てのAPIエンドポイント、パラメータ、スキーマ、スキーマプロパティ、リクエストボディ、レスポンスには `description` を記述すること。
- 全てのAPIエンドポイントの `description` には、処理内容を箇条書きで詳細に記載すること。（例：1. 入力のバリデーション 2. ユーザIDからデータ取得 3. ユーザレスポンスデータ生成）
- 全てのAPIエンドポイントには、`operationId` を記述すること。（例：`GetUser`、`CreateTask`）
